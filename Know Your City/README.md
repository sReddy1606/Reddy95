# Twitter Scraper Modernized (V1 & V2)

This repository contains the salvageable components of the original Master's project.

## Versions
- **V1 (Processor):** Core Python logic located in `src/processor.py`. It handles data sanitization and coordinate mapping.
- **V2 (UI):** Streamlit dashboard located in `src/app.py` for interactive data collection.

## Installation
1. Install Python 3.10+
2. Install dependencies: `pip install -r requirements.txt`
3. Run V2: `streamlit run src/app.py`
